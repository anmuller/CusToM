Huygens

PURPOSE

Computation of inertia matrix at point O

SYNOPSIS

function [Io] = Huygens(OG,Ig,m)

DESCRIPTION 

 Computation of inertia matrix at point O

   INPUT
   - OG: location of the center of gravity (G) from point O
   - Ig: inertia matrix at point G
   - m: solid mass
   OUTPUT
   - Io: inertia matrix at point O
